Detailed Description

This file defines the class Inet::IPAddress and related enumerated constants.

The CHIP Inet Layer uses objects of this class to represent Internet protocol addresses of both IPv4 and IPv6 address families. (IPv4 addresses are stored internally in the V4COMPAT format, reserved for that purpose.)

◆ ip4_addr_t

typedef ip_addr_t ip4_addr_t

Adaptation for LwIP ip4_addr_t type.

Before LwIP 2.0.0, the ip_addr_t type alias referred to a structure comprising an IPv4 address. At LwIP 2.0.0 and thereafter, this type alias is renamed ip4_addr_t and ip_addr_t is replaced with an alias to a union of both. Here, the ip4_addr_t type alias is provided even when the LwIP version is earlier than 2.0.0 so as to prepare for the import of the new logic.
