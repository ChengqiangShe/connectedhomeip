Detailed Description

This file contains declaration statements for CHIP System Layer-specific errors.

Error types, ranges, and mappings overrides may be made by defining the appropriate CHIP_SYSTEM_CONFIG_* or * CHIP_SYSTEM_CONFIG* macros.

NOTE WELL: On some platforms, this header is included by C-language programs.
